# Meshy Scripts

Codex should implement these TypeScript scripts during scaffold setup:

- `create-image-to-3d.ts`
- `create-multi-image-to-3d.ts`
- `poll-task.ts`
- `download-model.ts`

The scripts should follow `docs/05_MESHY_PIPELINE.md`.

Do not commit real API keys.
Do not print API keys.
Remember Meshy API calls may consume credits.
Scripts are dry-run by default; pass `--live` only after review.
